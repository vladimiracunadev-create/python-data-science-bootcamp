# 🎥 Presentación Online con Evaluador

## Guía Completa para 45 minutos

---

## 📋 Estructura de Evaluación

```
Evaluador está observando:
├── Claridad pedagógica (¿explicas bien?)
├── Conocimiento profundo (¿dominas el tema?)
├── Manejo de clase online (¿entiendes la plataforma?)
├── Timing (¿controlas los 45 min?)
├── Engagement (¿mantienes interés?)
└── Respuestas a preguntas (¿piensas rápido?)
```

---

## ⚙️ Configuración Técnica (15 min antes)

### **Antes de empezar:**

```
Pantalla:
  ✓ Cierra Slack, email, notificaciones
  ✓ Una sola pantalla abierta: tu presentación
  ✓ Zoom/Meet en modo presenter view
  ✓ Micrófono funciona (prueba 2 min antes)
  ✓ Cámara OK (frente a luz, sin contraluz)

Jupyter (si necesitas ejecutar código):
  ✓ Jupyter abierto en background (alt+tab)
  ✓ Código listo para ejecutar (NO escribas en vivo)
  ✓ Celda a celda (ejecuta paso a paso)

Pizarra virtual:
  ✓ Si vas a dibujar: Google Jamboard o igual preparado
  ✓ O usa un papel + cámara (personal, efectivo)
```

### **Ambiente de la sala:**

```
Físico:
  ✓ Habitación silenciosa
  ✓ Puerta cerrada
  ✓ Teléfono en silent
  ✓ Botella de agua a la mano
  ✓ Papel y lápiz para notas

Visual:
  ✓ Postura erguida (no recostado)
  ✓ Cámara a la altura de los ojos
  ✓ Luz de frente (no de atrás)
  ✓ Fondo neutral o profesional
```

---

## 🎬 Durante los 45 Minutos

### **Flujo exacto:**

```
00:00-01:00  Apertura y presentación (1 min)
             "Hola, hoy les mostraré..."
             
01:00-05:00  Contexto y por qué importa (4 min)
             "¿Por qué estudiamos esto?"
             
05:00-30:00  Contenido técnico (25 min)
             Slides + ejemplos
             
30:00-40:00  Ejercicio o demostración (10 min)
             Interactivo, en vivo
             
40:00-45:00  Resumen + preguntas (5 min)
             "Recapitulamos... ¿preguntas?"
```

---

## 💬 Comunicación Efectiva

### **Mientras hablas:**

✅ **HAZ:**
- Mira a la cámara (no a las notas)
- Habla claro y pausadamente
- Vocaliza bien (sin murmurar)
- Haz pausas después de puntos clave
- Sonríe (sí, en serio)
- Usa gestos naturales
- Mantén energía (no monotónico)

❌ **NO HAgas:**
- Leas las slides palabra por palabra
- Hables muy rápido (nervios)
- Digas "um", "uh", "eee" (rellenos)
- Mires hacia abajo (notas)
- Tapes la cámara
- Juegues con objetos (distrae)
- Uses jerga sin explicar

### **Si alguien interrumpe:**

```
Evaluador: "¿Qué significa [X]?"

TÚ: "Excelente pregunta. [Explica en 2-3 oraciones]
     ¿Queda claro? Continuamos..."

→ Demuestra que sabes pero también que manejas interrupciones
```

---

## 🎯 Timing (CRÍTICO)

### **Usa un reloj:**

```
Mientras hablas, mira el reloj cada 2 minutos
(discretamente, no obvio)

Si vas lento:
  → Acorta el ejercicio final
  
Si vas rápido:
  → Expande ejemplos
  → Haz más preguntas a los alumnos
  
Meta: Terminar entre 43-45 min (no antes, no después)
```

### **Puntos de control:**

```
Minuto 5: ¿Cubriste el "por qué"?
Minuto 15: ¿A la mitad del contenido?
Minuto 30: ¿Listo para ejercicio?
Minuto 40: ¿Resumen claro?
Minuto 45: FIN
```

## 🔧 Demos / Código Ejecutable

### **Si ejecutas código:**

```
ANTES:
  • Abre Jupyter en background
  • Todas las celdas listas (NO escribas en vivo)
  • Prueba 1 min antes que funcione

DURANTE:
  • "Veamos cómo funciona..." [Alt+Tab a Jupyter]
  • Ejecuta 1-2 celdas (no todo de golpe)
  • Explica el output
  • Vuelve a la presentación

TIMING:
  • Máx 3-4 min de demos (come tiempo)
  • Usa screenshots si es más rápido
```

### **Si algo falla:**

```
No te asustes. Es NORMAL.

RESPUESTA:
1. Reconoce el error (no lo ignores)
2. "Déjame reintentar..." [2-3 segundos]
3. Si persiste: "Mostraré una screenshot preparada..."
   [tienes backup, ¿verdad?]
4. Continúa como si nada

→ El evaluador valora que manejes presión
```

## 📝 Tres Opciones Disponibles

### **Opción 1: Visualización (Clase 6)**
- Dificultad: Media
- Impacto visual: ALTO
- Engagement: Bueno (gráficos bonitos)
- Pros: Demuestra creatividad
- Contras: Menos "hardcore" técnico

### **Opción 2: ML Intro (Clase 9)**
- Dificultad: Media-Alta
- Impacto visual: Medio
- Engagement: Bueno (conceptos fundamentales)
- Pros: Demuestra conceptos sólidos
- Contras: Riesgo de ser genérico

### **Opción 3: Pipelines (Clase 11)**
- Dificultad: Alta
- Impacto visual: Bajo
- Engagement: Alto si ejecutas código
- Pros: Demuestra profundidad y productividad
- Contras: Puede ser densa

**Recomendación:** Elige la que más domines. El evaluador ve que confíes en lo que hablas.

## 🎤 Preguntas Esperadas + Respuestas Rápidas

### **Preguntas genéricas:**

```
P: "¿Cómo enseñarías esto a alguien sin background?"
R: "Empezaría con [analogía simple], luego subiría
    gradualmente la complejidad. Siempre ejemplos antes
    de teoría."

P: "¿Cuál es el error más común en este tema?"
R: "[Error específico], lo vi cuando... [anécdota corta]"

P: "¿Cómo evalúas si un alumno entendió?"
R: "Hago ejercicios durante la clase y pregunto
    '¿alguien tiene dudas?' Escucho activamente."

P: "¿Por qué este tema importa en producción?"
R: "[Caso de uso real]... sin esto, pasaría [problema]."
```

### **Preguntas técnicas:**

Dependen de tu clase, pero sé honesto:
```
Si no sabes la respuesta:
  "Excelente pregunta. No lo sé de memoria, pero 
   buscaría [dónde]. La próxima clase lo traería."

→ Honestidad > adivinar
```

## 🚀 Mentalidad Final

```
Recuerda:

1. EL EVALUADOR QUIERE QUE TENGAS ÉXITO
   → No es un enemigo, es juez justo

2. TÚ DOMINAS ESTO
   → Has estudiado 8+ horas
   → Sabes el tema

3. 45 MIN ES CORTO
   → No intentes cubrir TODO
   → Cubre bien lo que cubre

4. LA CLARIDAD ES LA META
   → No la complejidad
   → Explica como si hablas con amigo inteligente

5. CONFIANZA (pero no arrogancia)
   → Seguridad al hablar
   → Humildad si no sabes algo
```

## 📊 Rúbrica (Lo que el evaluador ve)

| Criterio | ❌ Insuficiente | ⚠️ Regular | ✅ Bueno | 🌟 Excelente |
|----------|---|---|---|---|
| Claridad | Confuso | Algunas partes confusas | Claro | Crystal clear |
| Conocimiento | Errores graves | Algunos errores | Correcto | Profundo |
| Timing | Mucho menos/más | ±3 min | ±1 min | 45 min exacto |
| Engagement | Monótono | Plano | Bueno | Muy bueno |
| Respuestas | Evasivas | Superficiales | Buenas | Excelentes |
| Manejo online | Problemas técnicos | Algunas dificultades | Fluido | Profesional |

**Objetivo:** 4+ en la mayoría de criterios
