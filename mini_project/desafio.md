# Desafío integrador — Análisis de ventas

## Contexto

Una pequeña empresa quiere entender el desempeño de sus sucursales y categorías de productos a partir de un archivo CSV con ventas.

## Objetivo

Responder preguntas simples pero relevantes usando Python y pandas.

## Preguntas orientadoras

1. ¿Qué sucursal tiene mayor total neto vendido?
2. ¿Qué categoría concentra mayor ingreso?
3. ¿Qué vendedor acumula más ventas netas?
4. ¿Qué medio de pago aparece con mayor frecuencia?
5. ¿Existen productos con descuentos elevados que valga la pena revisar?

## Entregable sugerido

Un notebook que incluya:

- carga del dataset;
- limpieza o chequeos básicos;
- cálculos de apoyo;
- al menos dos tablas resumen;
- al menos un gráfico;
- conclusiones finales en lenguaje simple.

## Desafío opcional

Construir una recomendación breve para gerencia usando los hallazgos obtenidos.
