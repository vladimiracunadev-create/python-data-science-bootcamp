# Desafio integrador de analisis de ventas

> Desafio integrador para unir carga, analisis, visualizacion e interpretacion.

## Contexto

Una pequena empresa quiere entender el desempeno de sus sucursales y categorias de productos a partir de un archivo CSV con ventas.

## Objetivo

Responder preguntas simples pero relevantes usando Python y pandas.

## Preguntas orientadoras

1. Que sucursal tiene mayor total neto vendido?
2. Que categoria concentra mayor ingreso?
3. Que vendedor acumula mas ventas netas?
4. Que medio de pago aparece con mayor frecuencia?
5. Existen productos con descuentos elevados que valga la pena revisar?

## Entregable sugerido

Un notebook que incluya:

- carga del dataset;
- limpieza o chequeos basicos;
- calculos de apoyo;
- al menos dos tablas resumen;
- al menos un grafico;
- conclusiones finales en lenguaje simple.

## Desafio opcional

Construir una recomendacion breve para gerencia usando los hallazgos obtenidos.
