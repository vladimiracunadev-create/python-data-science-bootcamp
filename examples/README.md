# 🧪 Examples

> 🧪 Ejemplos rapidos para validar entorno, flujo y narrativa del bootcamp.


Contiene materiales de demostración rápida para validar el entorno y mostrar el enfoque del bootcamp sin recorrer todas las clases.

## 🧰 Archivos

- `notebook-demo.ipynb`: recorrido corto de análisis.
- `run_demo.py`: ejecución por consola.
- `validate_bootcamp.py`: validación mínima del entorno y datasets.
