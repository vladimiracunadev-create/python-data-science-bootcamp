Presentacion institucional

Una base de capacitacion tecnica con criterio pedagogico y crecimiento real

Este producto no es una coleccion de archivos. Es una base reusable para ensenar Python y Data Science en contexto escolar, con progresion de clases, practica guiada, portal del alumno, laboratorio interactivo y una operacion que puede crecer sin rehacerse desde cero.

Lo que ya existe hoy

  • curriculum modular con 12 clases;
  • datasets, ejercicios, notebooks y soluciones;
  • entorno Flask para practica y runner local;
  • GitHub Pages para alumnos;
  • health checks, CI y postura de seguridad visible.

La primera version escolar no necesita usar todo. El valor esta en poder empezar acotado y crecer sin perder estructura.

Operacion y seguridad

El demo ya muestra una forma responsable de construir

Seguridad visible

  • headers HTTP de seguridad en la app;
  • validacion de entradas y limites de payload;
  • timeout por ejecucion y reinicio de sesiones;
  • documentacion `SECURITY.md` y `RUNBOOK.md`.

Operacion reproducible

  • health endpoint y readiness endpoint;
  • arranque local por entorno virtual o Docker;
  • `docker compose` local y perfil endurecido;
  • portal del alumno publicado con GitHub Pages.

Validacion continua

  • workflow de CI para lint, tests y build de contenedor;
  • workflow de seguridad para `pip-audit` y `bandit`;
  • suite automatizada para endpoints y motor base.
