# ✨ Valor diferencial del enfoque

Este repositorio demuestra una combinacion de tres capas:

1. **Capa pedagogica**: objetivos, secuencia, pauta y evaluacion.
2. **Capa practica**: notebooks, datasets y scripts probables de usar en clase.
3. **Capa operativa**: entorno simple para que los ejercicios funcionen en el mismo ambiente.

No es solo material de ejemplo. Es una base reutilizable para impartir clases de manera ordenada y adaptable.
