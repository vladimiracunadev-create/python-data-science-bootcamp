# Bootcamp Python para Data Science

## Demo de mi sistema de capacitacion

### Adaptado para Skillnest / Colegio San Nicolas de Maipu

Entrevista: jueves 9 de abril de 2026, 15:30

## 1. Que traigo hoy

- Un repositorio docente ya funcional.
- Una muestra concreta de como diseno capacitaciones.
- Una propuesta V1 aterrizada al colegio.
- Una base que puede crecer aunque esta oportunidad no avance.

## 2. Que demuestra este demo sobre mi forma de trabajar

- Trabajo con progresion pedagogica, no con contenido suelto.
- Conecto explicacion, practica, evaluacion y cierre.
- Puedo adaptar el nivel sin perder estructura.
- Ya tengo una base reutilizable para distintas implementaciones.

## 3. Por que esto sigue teniendo valor aunque existan nuevas tecnologias

- La tecnologia puede asistir, pero no secuenciar aprendizaje por si sola.
- No detecta confusion, copia sin comprension o miedo a equivocarse.
- No convierte una respuesta en criterio, interpretacion y confianza.
- Mi valor esta en la mediacion pedagogica y en el diseno de la experiencia.

## 4. Que ya resuelve este repositorio

- clase 0 diagnostica + 12 clases de 90 minutos con objetivos, notebooks, ejercicios y soluciones.
- Datasets sinteticos listos para usar.
- Documentacion de metodologia, evaluacion y guia docente.
- App local para practicar Python dentro del mismo proyecto.
- Landing publica para alumnos en GitHub Pages y ruta clara de evolucion a movil.

## 5. Como lo acotaria para una primera version escolar

- Foco en Python, pandas y visualizacion basica.
- Una sola ruta principal de aprendizaje.
- Practica guiada antes que complejidad tecnica.
- Machine Learning y pipelines quedan para fase 2.

## 6. Ruta V1 recomendada

1. Fundamentos de Python aplicados a datos.
2. Lectura de CSV con pandas.
3. Filtros, columnas y tablas resumen.
4. Graficos simples e interpretacion.
5. Mini proyecto guiado.
6. Presentacion breve de hallazgos.

## 7. Como trabajaria en sala y que problemas tengo previstos

- Bloques de 90 minutos.
- Explicacion breve.
- Demostracion guiada.
- Ejercicio individual o en parejas.
- Cierre con retroalimentacion.
- Manejo de ritmos distintos, errores de sintaxis y uso de IA con reglas claras.

## 8. Como crece esta base

- De demo breve a piloto acotado.
- De piloto a bootcamp completo.
- De colegio a empresa o programa social con la misma base.
- De muestra tecnica a sistema de capacitacion reusable.

## 9. Que necesito confirmar para ejecutar bien

- Horas reales por semana.
- Fecha de inicio efectiva.
- Nivel y cantidad de estudiantes.
- Equipamiento disponible.
- Alcance pagado de preparacion, evaluacion y adaptacion.

## 10. Mi valor para este proyecto

- Puedo traducir contenido tecnico a aprendizaje claro.
- Ya tengo una base docente real, no una idea en abstracto.
- Puedo adaptar el material con criterio sin improvisar.
- Priorizo una primera implementacion ejecutable, no una promesa inflada.
