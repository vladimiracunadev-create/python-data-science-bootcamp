# 📚 Syllabus — Bootcamp Python para Data Science

> **31 clases · 13 módulos · ~46 horas · 30 notebooks · 6 datasets · 2 proyectos integradores**

---

## 🎯 Propósito general

Construir una base sólida y progresiva en Ciencia de Datos usando Python como herramienta principal. El programa cubre desde los fundamentos de programación hasta machine learning avanzado, procesamiento de texto, series de tiempo, ética en IA y despliegue de modelos. Está diseñado para ser accesible a cualquier edad y nivel de entrada, con ejemplos concretos, práctica guiada y proyectos reales.

## 👥 Perfil de entrada

- Personas sin experiencia previa o con base muy inicial en programación.
- Estudiantes de cualquier edad: desde secundaria hasta universitarios y profesionales.
- Participantes que valoran ejemplos concretos, retroalimentación continua y aprendizaje progresivo.

## 🏁 Perfil de salida esperado

Al finalizar, la persona podrá:

1. leer y escribir código Python con confianza;
2. usar NumPy y pandas para manipular y analizar datos;
3. consultar bases de datos con SQL desde Python;
4. construir gráficos claros e interpretativos (matplotlib y seaborn);
5. aplicar estadística descriptiva e inferencial a problemas reales;
6. diseñar y evaluar modelos de machine learning supervisados y no supervisados;
7. trabajar con texto (NLP), series de tiempo, anomalías y redes neuronales;
8. entender los principios éticos del uso de datos;
9. desplegar modelos básicos como APIs con Flask;
10. desarrollar proyectos integradores con preguntas, evidencias y conclusiones.

## 💡 Criterio de diseño

El programa privilegia:
- claridad antes que complejidad;
- comprensión progresiva con metáforas accesibles;
- datasets fáciles de explicar con contexto real;
- actividades ejecutables en el mismo entorno del curso;
- preguntas que conectan código con conclusiones reales.
