## 📌 Puntos que deben quedar claros
- Seaborn no reemplaza matplotlib, lo mejora: es una capa de abstracción con mejores valores por defecto.
- Cada tipo de gráfico responde una pregunta distinta: no todos los gráficos sirven para todo.
- `histplot` y `kdeplot` muestran cómo se distribuye una variable.
- `boxplot` y `violinplot` comparan distribuciones entre categorías y revelan outliers.
- `heatmap` muestra correlaciones entre muchas variables a la vez.
- `pairplot` es el explorador definitivo para ver todas las relaciones de un vistazo.
- El parámetro `hue` agrega una tercera dimensión visual usando color.
