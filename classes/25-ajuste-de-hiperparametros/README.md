# 📘 Clase 25: Ajuste de Hiperparámetros y Selección de Modelos

> 🎯 Ficha de clase con objetivo, materiales y foco de aprendizaje.

## 🎯 Objetivo
Comprender la diferencia entre parámetros e hiperparámetros, y aprender a usar GridSearchCV y RandomizedSearchCV para encontrar la configuración óptima de un modelo de Machine Learning, evaluando resultados con validación cruzada.

## ⏱️ Duración sugerida
90 minutos

## 📦 Dataset base
- `datasets/estudiantes.csv`

## ✅ Resultados esperados
Al finalizar, el estudiante podrá:
- Distinguir entre parámetros e hiperparámetros de un modelo, usar GridSearchCV y RandomizedSearchCV para optimizar modelos, interpretar curvas de aprendizaje y validación.

## 🧭 Temas clave
- Hiperparámetros vs parámetros
- GridSearchCV y RandomizedSearchCV
- Validación cruzada (cross-validation)
- Curvas de aprendizaje y de validación
- best_params_ y best_score_
- Optimización de RandomForest

- Cada bloque debe responder una pregunta concreta.

## 💡 Idea central
El ajuste de hiperparámetros es como calibrar un instrumento — pequeños cambios en la configuración pueden mejorar significativamente los resultados.

## 👩‍🏫 Nota para el docente
Usa la analogía del horno: la receta (modelo) puede ser perfecta, pero si el horno está a la temperatura equivocada (hiperparámetro incorrecto), el resultado no será bueno. Anima a los estudiantes a experimentar con distintos valores y observar cómo cambia el score.
