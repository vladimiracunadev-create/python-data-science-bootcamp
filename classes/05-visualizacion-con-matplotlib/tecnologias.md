# 🛠️ Tecnologías complementarias — Clase 05: Visualización con matplotlib

> Herramientas y librerías que complementan o extienden lo visto en esta clase.

## 📦 Librerías principales usadas en esta clase

| Librería | Versión recomendada | Para qué se usa en esta clase |
|---|---|---|
| `matplotlib` | >= 3.7 | Crear figuras controladas, definir ejes, rotular y ajustar gráficos |
| `pandas` | >= 2.0 | Leer el dataset y proveer las series de datos que se grafican |
| `numpy` | >= 1.24 | Operaciones numéricas auxiliares para generar datos de prueba o calcular rangos |

## 🔗 Herramientas complementarias

| Herramienta | Descripción | Cuándo usarla |
|---|---|---|
| `seaborn` | Librería de visualización estadística construida sobre matplotlib | Cuando se necesitan gráficos estadísticos (boxplot, heatmap, distribuciones) con menos código |
| `plotly` | Visualizaciones interactivas en el navegador | Cuando el gráfico será visto en un dashboard o necesita zoom y tooltips |
| `mplcursors` | Agrega tooltips interactivos a gráficos de matplotlib | Cuando se quiere inspeccionar puntos individuales en un gráfico estático |
| Jupyter Notebook / JupyterLab | Entorno donde los gráficos se renderizan inline | Para presentar el análisis con código y resultado juntos |
