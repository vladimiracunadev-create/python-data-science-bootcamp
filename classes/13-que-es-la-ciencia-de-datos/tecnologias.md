# 🔧 Tecnologías complementarias — Clase 13: ¿Qué es la Ciencia de Datos?

> Herramientas, librerías y recursos para ampliar lo aprendido.

## 📦 Librerías relacionadas

| Librería | Para qué sirve | Nivel sugerido |
|---|---|---|
| `pandas` | Manipulación y análisis de datos tabulares con DataFrames | Básico |
| `numpy` | Cálculo numérico rápido con arrays y álgebra lineal | Básico |
| `matplotlib` | Visualización básica: gráficos de líneas, barras, dispersión | Básico |
| `scikit-learn` | Machine learning completo: modelos, evaluación y pipelines | Intermedio |
| `seaborn` | Visualización estadística de alto nivel construida sobre matplotlib | Intermedio |
| `scipy` | Estadística avanzada, optimización y procesamiento de señales | Intermedio |
| `jupyter` | Entorno de notebooks interactivo para exploración de datos | Básico |
| `plotly` | Gráficos interactivos con zoom, tooltips y dashboards | Intermedio |
| `statsmodels` | Modelos estadísticos formales: regresión, series de tiempo | Avanzado |

## 🌐 Recursos recomendados

- **Documentación oficial**: pandas — [pandas.pydata.org/docs](https://pandas.pydata.org/docs/) — guía completa con ejemplos para manipulación de datos desde cero
- **Tutorial recomendado**: "Python for Data Science Handbook" de Jake VanderPlas (disponible gratis en línea) — cubre numpy, pandas, matplotlib y scikit-learn de forma integrada y progresiva
- **Concepto clave para buscar**: "ciclo de vida de un proyecto de datos" / "data science project lifecycle" — explica cómo se estructura el trabajo real en la industria

## 🚀 Próximos pasos sugeridos

- Aprender pandas en profundidad: filtrado condicional, agrupación, combinación de tablas y manejo de nulos
- Explorar visualización estadística con seaborn para entender distribuciones de datos antes de modelar
- Practicar con datasets reales de Kaggle o del repositorio UCI Machine Learning para aplicar el ciclo CRISP-DM completo
- Estudiar el proceso completo de un proyecto de machine learning con scikit-learn: desde cargar datos hasta evaluar un modelo

## 🧰 Herramientas alternativas

| Herramienta | Descripción breve | Cuándo conviene usarla |
|---|---|---|
| R + tidyverse | Lenguaje estadístico con librerías equivalentes a pandas y ggplot2 | Proyectos académicos o con fuerte componente de estadística formal |
| KNIME | Plataforma visual de ciencia de datos sin necesidad de código | Para usuarios no programadores que necesitan armar flujos de datos |
| Google Colab | Notebooks Jupyter gratuitos en la nube con acceso a GPU | Cuando no tienes Python instalado localmente o necesitas GPU para deep learning |
| Tableau / Power BI | Herramientas de Business Intelligence con visualizaciones interactivas | Dashboards ejecutivos y reportes para audiencias no técnicas |
