# 🛠️ Tecnologías complementarias — Clase 01: Fundamentos de Python aplicados a datos

> Herramientas y librerías que complementan o extienden lo visto en esta clase.

## 📦 Librerías principales usadas en esta clase

| Librería | Versión recomendada | Para qué se usa en esta clase |
|---|---|---|
| Python (stdlib) | 3.10+ | Variables, listas, diccionarios, funciones y control de flujo — todo el contenido central de la clase |
| `collections` | stdlib | `Counter` y `defaultdict` para agrupar y contar sin escribir bucles manuales |
| `math` | stdlib | Operaciones matemáticas básicas como redondeo, raíz cuadrada y valor absoluto |
| `pprint` | stdlib | Imprimir diccionarios y listas anidadas de forma legible durante el desarrollo |

## 🔗 Herramientas complementarias

| Herramienta | Descripción | Cuándo usarla |
|---|---|---|
| Jupyter Notebook | Entorno interactivo que ejecuta bloques de código independientes | Durante toda la clase para probar cada concepto y ver resultados inmediatos |
| Python Tutor (pythontutor.com) | Visualizador paso a paso de ejecución de código Python | Cuando el estudiante no entiende qué hace un bucle o cómo se asigna una variable |
| VS Code + extensión Python | Editor con autocompletado, linter y terminal integrada | Para escribir scripts más largos y practicar fuera del notebook |
| Flake8 / Ruff | Linter que detecta errores de estilo y sintaxis | Para instalar buenos hábitos de escritura de código desde la primera clase |

## 📚 Recursos para profundizar

- [Tutorial oficial de Python en español](https://docs.python.org/es/3/tutorial/): documentación oficial con ejemplos claros de variables, listas, diccionarios y funciones
- [Automate the Boring Stuff with Python](https://automatetheboringstuff.com/): recurso para profundizar en aplicaciones prácticas de Python sin conocimiento previo
- [Real Python — Python Basics](https://realpython.com/learning-paths/python-basics/): guía práctica con ejercicios interactivos sobre los temas de esta clase
