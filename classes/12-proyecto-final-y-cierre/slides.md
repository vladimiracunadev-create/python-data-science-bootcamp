# Slides — Clase 12: Proyecto Final y Cierre del Bootcamp

## Bloque 1 (20 min) — Estructura de un proyecto de Data Science profesional

```
📁 mi_proyecto/
├── datos/          ← datos originales, sin modificar
├── notebooks/      ← análisis exploratorio y modelado
├── src/            ← código reutilizable
├── resultados/     ← outputs, gráficos, modelos guardados
└── README.md       ← descripción del proyecto
```

**Principio clave:** un proyecto reproducible es un proyecto profesional.

---

## Bloque 2 (30 min) — Checklist del proyecto completo

### 1. Exploración (EDA)
- [ ] Dimensiones del dataset
- [ ] Tipos de datos y columnas
- [ ] Distribuciones de variables clave
- [ ] Correlaciones relevantes
- [ ] Outliers identificados

### 2. Limpieza
- [ ] Valores nulos manejados
- [ ] Tipos de datos correctos
- [ ] Duplicados eliminados
- [ ] Variables creadas si es necesario

### 3. Modelado
- [ ] Train/test split realizado
- [ ] Pipeline construido
- [ ] Al menos 2 modelos comparados
- [ ] Métricas reportadas

### 4. Comunicación
- [ ] Al menos 3 visualizaciones
- [ ] Conclusiones en lenguaje de negocio
- [ ] Limitaciones del análisis explicitadas

---

## Bloque 3 (40 min) — Presentaciones (5 min por estudiante)

Estructura sugerida:
1. **¿Qué problema resolví?** (30 seg)
2. **¿Qué encontré en los datos?** (1 min)
3. **¿Qué modelo usé y cuán bueno es?** (1 min)
4. **¿Qué recomendarías al negocio?** (1 min)
5. **¿Qué harías diferente?** (30 seg)

## Cierre del Bootcamp (20 min)

**Lo que aprendiste en 12 clases:**

| Semana | Habilidades |
|---|---|
| 1-2 | Python básico, estructuras de datos |
| 3-4 | Pandas, limpieza, transformación |
| 5-6 | Visualización, estadística descriptiva |
| 7-8 | Proyectos guiados, presentación de datos |
| 9-10 | Machine Learning: regresión y clasificación |
| 11-12 | Evaluación robusta, pipelines, proyecto final |

**Próximos pasos sugeridos:**
- Kaggle (competencias de práctica)
- Proyecto propio con datos reales
- Especialización en ML avanzado o MLOps
