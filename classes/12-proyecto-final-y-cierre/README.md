# Clase 12 — Proyecto Final y Cierre del Bootcamp

## Objetivo

Integrar todos los aprendizajes del bootcamp en un análisis completo: desde la carga y limpieza de datos hasta la presentación de un modelo y sus conclusiones.

## Duración sugerida

120 minutos

## Dataset base

A elección del estudiante (cualquiera de los 5 datasets del bootcamp)

## Resultados esperados

Al finalizar, el estudiante habrá:

- realizado un análisis exploratorio completo sobre un dataset real;
- limpiado y transformado los datos de forma reproducible;
- construido y evaluado al menos un modelo predictivo;
- visualizado los hallazgos clave con gráficos explicativos;
- presentado sus conclusiones de forma clara para audiencia no técnica.

## Materiales

- `notebook.ipynb` — template guiado del proyecto
- `slides.md` — pauta de la clase y criterios de evaluación
- `ejercicios.md` — checklist de entregables
- `homework.md` — instrucciones de entrega final
- `soluciones.ipynb` — ejemplo de proyecto completo resuelto
- `teoria.md` — guía de presentación de resultados

## Estructura del proyecto final

```
1. Descripción del problema y el dataset (5 min)
2. Análisis exploratorio: estadísticas + visualizaciones (20 min)
3. Limpieza y transformación de datos (15 min)
4. Modelado y evaluación (20 min)
5. Conclusiones e insights para el negocio (10 min)
```

## Rúbrica resumida

| Criterio | Peso |
|---|---|
| Claridad del análisis exploratorio | 25% |
| Calidad del preprocesamiento | 20% |
| Construcción y evaluación del modelo | 25% |
| Visualizaciones comunicativas | 15% |
| Conclusiones orientadas al negocio | 15% |

## Nota para el docente

Este es el momento de celebrar el progreso. Cada estudiante llegó sin saber Python; ahora puede construir un análisis completo. Dar feedback positivo específico a cada uno antes del cierre formal.
